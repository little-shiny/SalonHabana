SalonHabana: DI01 DAM

Este proyecto se ha creado para resolver el ejercicio propuesto en Desarrollo de Interfaces de DAM. El proyecto consta de una interfaz gráfica mediante la cual el usuario puede solicitar una reserva de un salón de eventos con diferentes opciones y características. Consta de una interfaz que utilizando un CardLayout se muestran diferentes JPanels con el contenido para el usuario mediante la navegación guiada con botones. El proyecto contiene tres JPanels principales:

  • Panel de inicio: en él se muestran dos botones para navegar a los otros dos paneles
  • Panel de reservas: permiten al usuario introducir sus datos personales con una validación de los mismos y acceder a una pestaña con los datos de la reserva como el tipo del evento, los invitados, el tipo de comida... Para los cuales se han utilizado diferentes elementos de Java Swing como los RaidButtons o los Spinners.
  • Panel de contacto: se mostrarían los datos de contacto de la empresa junto con un botón que permite volver al menú principal

Otras mejoras implementadas

Se han añadido otras mejoras como son el paquete ValidateEntry, que contiene ciertas clases para validar las entradas del usuario y que no permitan escribir letras en el número de teléfono, por ejemplo. Se han utilizado expresiones regulares para ello. Además, para poder mostrar y ocultar las opciones adicionales cuando el usuario selecciona la opción de "congreso" he creado lo que he llamado "panel dinámico", que básicamente oculta y activa el contenido en función de la selección del RaidButton.
